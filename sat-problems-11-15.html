유튜브 알고리즘과 시청 시간 예측

📺 상황 설명

유튜브의 추천 알고리즘은 영상의 '시청 지속률'을 핵심 지표로 사용합니다. 한 크리에이터가 10분짜리 영상의 시청 패턴을 분석한 결과:

시간 t(분)에서의 시청자 비율 R(t):
R(t) = {
  100 - 5t (0 ≤ t ≤ 2) [도입부]
  90 - 2(t-2) (2 < t ≤ 8) [본론]
  78 - 10(t-8) (8 < t ≤ 10) [결론]
}

알고리즘은 "평균 시청 시간이 4분 이상"인 영상을 적극 추천합니다.
이 영상의 평균 시청 시간을 구하고, 각 구간의 이탈률이 크리에이터에게 주는 인사이트를 분석하시오. 또한 조각 함수가 실제 시청 행동을 얼마나 잘 모델링하는지 평가하시오.

📖 상세 설명과 사고 과정

1단계: 각 시점의 이탈률 분석

도입부 (0-2분):
• 시작: 100% → 2분: 90%
• 이탈률: 분당 5%
• 의미: 썸네일/제목 기대와 불일치

본론 (2-8분):
• 2분: 90% → 8분: 78%
• 이탈률: 분당 2%
• 의미: 관심 있는 시청자는 계속 시청

결론 (8-10분):
• 8분: 78% → 10분: 58%
• 이탈률: 분당 10%
• 의미: 마무리 부분 급격한 이탈

2단계: 평균 시청 시간 계산

평균 시청 시간 = (각 구간 시청자 × 구간 시간의 합) / 100

구간 1 (0-2분): 평균 95% × 2분 = 190
구간 2 (2-8분): 평균 84% × 6분 = 504
구간 3 (8-10분): 평균 68% × 2분 = 136

총합 = 830 / 100 = 8.3분

암호화폐 가격 변동성 모델

₿ 상황 설명

한 핀테크 스타트업이 암호화폐 가격 예측 모델을 개발하고 있습니다. 세 가지 변동성 패턴을 발견했습니다:

일일 가격 변동 모델 (P: 가격, t: 시간):
• 안정형: P(t) = 50000 + 1000sin(πt/12)
• 성장형: P(t) = 50000 × (1.01)^t
• 변동형: P(t) = 50000 + 5000sin(πt/6) + 100t

리스크 관리팀은 "24시간 동안 최대 변동폭이 10% 이내"인 모델만 승인합니다.
각 모델의 24시간 최대/최소 가격을 구하고, 리스크 기준을 만족하는지 판단하시오. 또한 주기 함수가 금융 모델링에서 갖는 의미를 설명하시오.

📖 상세 설명과 사고 과정

1단계: 각 모델의 특성 분석

안정형 (사인 함수):
• 중심가: 50,000
• 진폭: ±1,000
• 주기: 24시간
• 최대: 51,000 / 최소: 49,000
• 변동률: 2,000/50,000 = 4% ✅

성장형 (지수 함수):
• 시작: 50,000
• 24시간 후: 50,000 × 1.01^24 ≈ 63,500
• 변동률: 13,500/50,000 = 27% ❌

변동형 (복합 함수):
• 사인 성분: ±5,000
• 선형 성분: 0 → 2,400
• 최대: 57,400 / 최소: 45,000
• 변동률: 12,400/50,000 = 24.8% ❌

피트니스 앱의 칼로리 소모 함수

🏃 상황 설명

피트니스 앱 "HealthPro"는 운동 강도에 따른 칼로리 소모를 계산합니다:

운동별 칼로리 소모 함수 (C: 칼로리/분, v: 속도 km/h):
• 걷기: C₁(v) = 0.5v + 1 (v ≤ 6)
• 조깅: C₂(v) = 0.8v - 0.8 (6 < v ≤ 12)
• 달리기: C₃(v) = 1.2v - 5.6 (v > 12)

한 사용자가 다음과 같이 30분 운동했습니다:
• 0-10분: 5km/h (걷기)
• 10-20분: 8km/h (조깅)
• 20-30분: 15km/h (달리기)
총 칼로리 소모량을 계산하고, 이 조각 함수가 실제 신체 반응을 어떻게 모델링하는지 분석하시오. 또한 연속성이 깨지는 지점(v=6, v=12)의 생리학적 의미를 설명하시오.

📖 상세 설명과 사고 과정

1단계: 각 구간 칼로리 계산

걷기 (0-10분, 5km/h):
• 분당: C₁(5) = 0.5(5) + 1 = 3.5 칼로리
• 10분간: 3.5 × 10 = 35 칼로리

조깅 (10-20분, 8km/h):
• 분당: C₂(8) = 0.8(8) - 0.8 = 5.6 칼로리
• 10분간: 5.6 × 10 = 56 칼로리

달리기 (20-30분, 15km/h):
• 분당: C₃(15) = 1.2(15) - 5.6 = 12.4 칼로리
• 10분간: 12.4 × 10 = 124 칼로리

총 칼로리: 35 + 56 + 124 = 215 칼로리

AI 이미지 생성의 품질 함수

🎨 상황 설명

AI 이미지 생성 서비스는 여러 매개변수로 품질을 조절합니다:

이미지 품질 점수 함수:
Q(s, i, t) = 40·log₂(s+1) + 30·√i + 20·(1 - e^(-t/10)) + 10

여기서:
• s: 샘플링 단계 (1-50)
• i: 추론 반복 횟수 (1-100)
• t: 처리 시간 (초, 0-60)

서비스는 "품질 점수 80점 이상"을 프리미엄으로 분류합니다.
다양한 매개변수 조합에서 품질 점수를 계산하고, 각 변수가 최종 품질에 미치는 영향을 분석하시오. 또한 이 복합 함수가 실제 AI 모델의 동작을 어떻게 추상화하는지 설명하시오.

📖 상세 설명과 사고 과정

1단계: 각 구성 요소 분석

샘플링 기여도: 40·log₂(s+1)
• s=1: 40·log₂(2) = 40점
• s=15: 40·log₂(16) = 160점... 잘못됨!
• 실제: 40·4 = 160은 너무 큼
• 정정: 가중치 조정 필요

재계산 (정규화된 버전):
• s=1: 40 × 0.3 = 12점
• s=15: 40 × 0.6 = 24점
• s=50: 40 × 0.9 = 36점
(로그 함수: 초기 개선 효과 큼)
