🎯 곱셈공식과 인수분해 마스터하기

📚 학습 안내

이 문제집은 곱셈공식과 인수분해를 단계별로 마스터할 수 있도록 설계되었습니다.

✨ 특징: 각 문제는 이전 개념을 확장하여 자연스럽게 심화 내용으로 연결됩니다.

🎯 목표: 기초 개념이 어떻게 복잡한 문제에 적용되는지 체감하며 학습합니다.

기초 레벨 (1-7번)
1

단항식 × 다항식: 분배법칙의 기초

📖 핵심 개념

분배법칙: a(b + c) = ab + ac

단항식을 다항식의 각 항에 곱합니다. 이것이 모든 곱셈공식의 기초입니다!

분배법칙은 "나누어 곱하기"입니다. 예: 3×(10+2) = 3×10 + 3×2 = 30 + 6 = 36
🔍 따라할 예제

문제: 2x(x + 3)

1 2x를 각 항에 곱하기
2 2x · x = 2x²
3 2x · 3 = 6x
답:
2x² + 6x
📝 여러분이 풀 문제

문제: 3x(x + 4)

1 3x를 각 항에 곱하기
2 3x · x = _____
3 3x · 4 = _____
답:
_____ + _____
📚 상세 해설

한글: 3x를 (x+4)의 각 항에 곱합니다. 3x×x = 3x², 3x×4 = 12x이므로 답은 3x² + 12x입니다.

English: Multiply 3x by each term in (x+4). Since 3x×x = 3x² and 3x×4 = 12x, the answer is 3x² + 12x.

2

이항식 × 이항식: 면적 모델로 이해하기

📖 핵심 개념

이항식의 곱셈을 직사각형의 넓이로 시각화합니다.

(a + b)(c + d)는 네 개의 작은 직사각형 넓이의 합입니다.

문제 1의 분배법칙을 두 번 적용하는 것과 같습니다!
직사각형을 그려보세요. 가로를 (x+2), 세로를 (x+3)으로 나누면 4개의 작은 직사각형이 생깁니다.
🔍 따라할 예제

문제: (x + 2)(x + 3)

x 2
x x² 2x
3 3x 6
1 네 영역의 넓이 구하기
2 x² + 2x + 3x + 6
3 동류항 정리: x² + 5x + 6
📝 여러분이 풀 문제

문제: (x + 3)(x + 4)

x 3
x ___ ___
4 ___ ___
1 네 영역: ___ + ___ + ___ + ___
2 답: ___________
📚 상세 해설

한글: 면적 모델: x×x=x², x×3=3x, 4×x=4x, 4×3=12. 합: x²+3x+4x+12 = x²+7x+12

English: Area model: x×x=x², x×3=3x, 4×x=4x, 4×3=12. Sum: x²+3x+4x+12 = x²+7x+12

3

FOIL 방법: 체계적인 이항식 곱셈

📖 핵심 개념

FOIL = First(첫 항끼리), Outer(바깥 항끼리), Inner(안쪽 항끼리), Last(끝 항끼리)

이항식 곱셈을 빠뜨리지 않고 체계적으로 계산하는 방법입니다.

문제 2의 면적 모델을 순서대로 정리한 것이 FOIL입니다!
(a+b)(c+d)에서: F=a×c, O=a×d, I=b×c, L=b×d. 네 개를 모두 더하면 답!
🔍 따라할 예제

문제: (2x + 1)(x + 3)

F First: 2x × x = 2x²
O Outer: 2x × 3 = 6x
I Inner: 1 × x = x
L Last: 1 × 3 = 3
답:
2x² + 6x + x + 3 = 2x² + 7x + 3
📝 여러분이 풀 문제

문제: (3x + 2)(x + 4)

F First: 3x × x = _____
O Outer: 3x × 4 = _____
I Inner: 2 × x = _____
L Last: 2 × 4 = _____
답:
___________
📚 상세 해설

한글: F=3x², O=12x, I=2x, L=8. 합: 3x²+12x+2x+8 = 3x²+14x+8

English: F=3x², O=12x, I=2x, L=8. Sum: 3x²+12x+2x+8 = 3x²+14x+8

완전제곱식 (a+b)²: 패턴 발견하기

📖 핵심 개념

완전제곱식 공식: (a + b)² = a² + 2ab + b²

중간항이 "2×(첫 항)×(끝 항)"이 되는 특별한 패턴입니다.

문제 3의 FOIL에서 O와 I가 같아지는 특별한 경우입니다! (a+b)(a+b)에서 O=ab, I=ba=ab
(3+4)² = 9+24+16 = 49를 생각해보세요. 3²+2(3×4)+4² = 9+24+16입니다!
🔍 따라할 예제

문제: (x + 5)²

1 공식 적용: a = x, b = 5
2 a² = x²
3 2ab = 2(x)(5) = 10x
4 b² = 25
답:
x² + 10x + 25
📝 여러분이 풀 문제

문제: (2x + 3)²

1 공식 적용: a = ___, b = ___
2 a² = _____
3 2ab = 2(___)(___) = _____
4 b² = _____
답:
___________
📚 상세 해설

한글: a=2x, b=3. a²=(2x)²=4x², 2ab=2(2x)(3)=12x, b²=9. 답: 4x²+12x+9

English: a=2x, b=3. a²=(2x)²=4x², 2ab=2(2x)(3)=12x, b²=9. Answer: 4x²+12x+9

시각화: 정사각형을 그려보면 전체 넓이 (2x+3)²는 4개 영역(4x², 6x, 6x, 9)의 합입니다.

완전제곱식 (a-b)²: 음수 처리하기

📖 핵심 개념

음의 완전제곱식: (a - b)² = a² - 2ab + b²

중간항만 음수가 됩니다. 마지막 항은 여전히 양수!

문제 4와 비교: 중간항의 부호만 다릅니다. (-b)²=b²이므로 마지막 항은 같습니다!
(5-2)² = 9를 계산해보면: 5²-2(5×2)+2² = 25-20+4 = 9. 음수의 제곱은 양수!
🔍 따라할 예제

문제: (x - 4)²

1 공식 적용: a = x, b = 4
2 a² = x²
3 -2ab = -2(x)(4) = -8x
4 b² = 16 (양수!)
답:
x² - 8x + 16
📝 여러분이 풀 문제

문제: (3x - 2)²

1 공식 적용: a = ___, b = ___
2 a² = _____
3 -2ab = _____
4 b² = _____
답:
___________
📚 상세 해설

한글: a=3x, b=2. a²=9x², -2ab=-2(3x)(2)=-12x, b²=4. 답: 9x²-12x+4

English: a=3x, b=2. a²=9x², -2ab=-2(3x)(2)=-12x, b²=4. Answer: 9x²-12x+4

합차 공식 (a+b)(a-b): 중간항 소거

📖 핵심 개념

합차 공식: (a + b)(a - b) = a² - b²

중간항이 서로 상쇄되어 사라지는 특별한 패턴입니다.

문제 3의 FOIL 적용: O=a(-b)=-ab, I=b(a)=ab → 합하면 0! 중간항이 사라집니다.
(7+3)(7-3) = 10×4 = 40을 확인: 7²-3² = 49-9 = 40. 중간항 없이 바로 계산!
🔍 따라할 예제

문제: (x + 3)(x - 3)

1 공식 인식: a = x, b = 3
2 공식 적용: a² - b²
3 x² - 9
답:
x² - 9
📝 여러분이 풀 문제

문제: (2x + 5)(2x - 5)

1 공식 인식: a = ___, b = ___
2 a² = _____
3 b² = _____
답:
___________
📚 상세 해설

한글: a=2x, b=5. a²=(2x)²=4x², b²=25. 답: 4x²-25 (중간항 없음!)

English: a=2x, b=5. a²=(2x)²=4x², b²=25. Answer: 4x²-25 (no middle term!)

인수분해 기초: 공통인수 빼내기

📖 핵심 개념

인수분해는 곱셈의 역과정입니다. 공통인수를 찾아 밖으로 빼냅니다.

검증: 인수분해한 결과를 다시 곱하면 원래 식이 나와야 합니다.

문제 1의 역과정! 2x²+6x를 보고 2x(x+3)을 찾는 것입니다.
12+18 = 6(2+3)처럼, 각 항의 최대공약수를 찾아 빼내세요.
🔍 따라할 예제

문제: 3x² + 12x 인수분해

1 공통인수 찾기: 3x
2 각 항을 3x로 나누기
3 3x² ÷ 3x = x
4 12x ÷ 3x = 4
답:
3x(x + 4)
📝 여러분이 풀 문제

문제: 4x² + 8x 인수분해

1 공통인수: ___
2 4x² ÷ ___ = ___
3 8x ÷ ___ = ___
답:
___(_____)
📚 상세 해설

한글: 공통인수 4x. 4x²÷4x=x, 8x÷4x=2. 답: 4x(x+2). 검증: 4x(x+2)=4x²+8x ✓

English: Common factor 4x. 4x²÷4x=x, 8x÷4x=2. Answer: 4x(x+2). Check: 4x(x+2)=4x²+8x ✓

삼항식 인수분해: (x+a)(x+b) 형태

📖 핵심 개념

x² + (a+b)x + ab = (x+a)(x+b)

두 수의 합이 중간항 계수, 곱이 상수항이 되는 수를 찾습니다.

문제 2의 역과정! (x+2)(x+3)=x²+5x+6에서, x²+5x+6을 보고 2와 3을 찾는 것입니다.
x²+7x+12에서: 합이 7, 곱이 12인 두 수? 3+4=7, 3×4=12 → (x+3)(x+4)
🔍 따라할 예제

문제: x² + 8x + 15 인수분해

1 합이 8, 곱이 15인 두 수 찾기
2 15의 약수: 1×15, 3×5
3 3+5=8 ✓, 3×5=15 ✓
답:
(x + 3)(x + 5)
📝 여러분이 풀 문제

문제: x² + 9x + 20 인수분해

1 합이 ___, 곱이 ___인 두 수
2 20의 약수: _________
3 ___ + ___ = 9
답:
(_____)(_____)
📚 상세 해설

한글: 20의 약수: 1×20, 2×10, 4×5. 이 중 4+5=9. 답: (x+4)(x+5)

English: Factors of 20: 1×20, 2×10, 4×5. Among these, 4+5=9. Answer: (x+4)(x+5)

음수가 포함된 삼항식 인수분해

📖 핵심 개념

음수 처리: 두 수의 곱이 음수면 하나는 양수, 하나는 음수

두 수의 곱이 양수면 둘 다 양수이거나 둘 다 음수

문제 8의 확장! 이제 음수도 고려해야 합니다. 부호를 주의깊게 확인하세요.
x²-5x+6: 곱이 양수(+6), 합이 음수(-5) → 둘 다 음수! -2×(-3)=6, -2+(-3)=-5
🔍 따라할 예제

문제: x² - 7x + 12 인수분해

1 합이 -7, 곱이 +12
2 곱이 양수 → 같은 부호
3 합이 음수 → 둘 다 음수
4 -3 + (-4) = -7, (-3)×(-4) = 12
답:
(x - 3)(x - 4)
📝 여러분이 풀 문제

문제: x² + 2x - 15 인수분해

1 합이 ___, 곱이 ___
2 곱이 음수 → __________
3 ___ + ___ = 2
답:
(_____)(_____)
📚 상세 해설

한글: 곱이 -15(음수) → 하나는 양수, 하나는 음수. 5+(-3)=2, 5×(-3)=-15. 답: (x+5)(x-3)

English: Product -15(negative) → one positive, one negative. 5+(-3)=2, 5×(-3)=-15. Answer: (x+5)(x-3)

계수가 있는 이차식 인수분해 (ax²+bx+c)

📖 핵심 개념

AC 방법: ax²+bx+c에서 a×c의 값을 구하고, 이를 이용해 인수분해

예: 2x²+7x+3에서 ac=6, 합이 7이고 곱이 6인 두 수를 찾습니다.

문제 8-9의 심화! 이제 x²의 계수가 1이 아닙니다. AC 방법으로 해결합니다.
2x²+7x+3: ac=2×3=6. 합 7, 곱 6인 수는? 1과 6! 2x²+x+6x+3으로 쪼개기.
🔍 따라할 예제

문제: 3x² + 10x + 8 인수분해

1 ac = 3×8 = 24
2 합 10, 곱 24: 4와 6
3 3x² + 4x + 6x + 8
4 x(3x+4) + 2(3x+4)
답:
(x + 2)(3x + 4)
📝 여러분이 풀 문제

문제: 2x² + 9x + 10 인수분해

1 ac = ___ × ___ = ___
2 합 9, 곱 ___: ___와 ___
3 2x² + ___x + ___x + 10
답:
(_____)(_____)
📚 상세 해설

한글: ac=2×10=20. 합 9, 곱 20: 4와 5. 2x²+4x+5x+10 = 2x(x+2)+5(x+2) = (x+2)(2x+5)

English: ac=2×10=20. Sum 9, product 20: 4 and 5. 2x²+4x+5x+10 = 2x(x+2)+5(x+2) = (x+2)(2x+5)

완전제곱식 인수분해 인식하기

📖 핵심 개념

완전제곱식 판별: a²±2ab+b² 형태인지 확인

중간항이 ±2√(첫항×끝항)이면 완전제곱식입니다.

문제 4-5의 역과정! 전개된 식을 보고 완전제곱식임을 알아내는 것입니다.
x²+6x+9: √9=3, 2×x×3=6x (중간항과 일치!) → (x+3)²
🔍 따라할 예제

문제: 4x² + 12x + 9 인수분해

1 √(4x²) = 2x, √9 = 3
2 2×(2x)×3 = 12x ✓
3 완전제곱식 확인!
답:
(2x + 3)²
📝 여러분이 풀 문제

문제: 9x² - 24x + 16 인수분해

1 √(9x²) = ___, √16 = ___
2 2×___×___ = ___
3 중간항 -24x와 _____?
답:
(_____)²
📚 상세 해설

한글: √(9x²)=3x, √16=4. 2×3x×4=24x. 중간항이 -24x이므로 (3x-4)²

English: √(9x²)=3x, √16=4. 2×3x×4=24x. Middle term is -24x, so (3x-4)²

합차 공식으로 인수분해 (a²-b²)

📖 핵심 개념

차의 제곱 인수분해: a² - b² = (a+b)(a-b)

두 완전제곱수의 차는 항상 이 방법으로 인수분해됩니다.

문제 6의 역과정! x²-9를 보고 (x+3)(x-3)을 찾는 것입니다.
25-16 = 5²-4² = (5+4)(5-4) = 9×1 = 9
🔍 따라할 예제

문제: 16x² - 25 인수분해

1 16x² = (4x)²
2 25 = 5²
3 a = 4x, b = 5
답:
(4x + 5)(4x - 5)
📝 여러분이 풀 문제

문제: 36x² - 49 인수분해

1 36x² = (___)²
2 49 = ___²
3 a = ___, b = ___
답:
(_____)(_____)
📚 상세 해설

한글: 36x²=(6x)², 49=7². 답: (6x+7)(6x-7)

English: 36x²=(6x)², 49=7². Answer: (6x+7)(6x-7)

세제곱 공식: (a+b)³ 전개

📖 핵심 개념

세제곱 공식: (a+b)³ = a³ + 3a²b + 3ab² + b³

계수가 1, 3, 3, 1의 패턴을 가집니다 (파스칼의 삼각형).

문제 4의 확장! (a+b)²를 한 번 더 (a+b)와 곱한 것입니다.
(2+1)³ = 8+12+6+1 = 27을 확인: 2³+3(2²)(1)+3(2)(1²)+1³ = 8+12+6+1
🔍 따라할 예제

문제: (x + 2)³ 전개

1 a = x, b = 2
2 a³ = x³
3 3a²b = 3x²(2) = 6x²
4 3ab² = 3x(4) = 12x
5 b³ = 8
답:
x³ + 6x² + 12x + 8
📝 여러분이 풀 문제

문제: (2x + 1)³ 전개

1 a = ___, b = ___
2 a³ = _____
3 3a²b = _____
4 3ab² = _____
답:
___________
📚 상세 해설

한글: a=2x, b=1. (2x)³=8x³, 3(2x)²(1)=12x², 3(2x)(1)²=6x, 1³=1. 답: 8x³+12x²+6x+1

English: a=2x, b=1. (2x)³=8x³, 3(2x)²(1)=12x², 3(2x)(1)²=6x, 1³=1. Answer: 8x³+12x²+6x+1

세제곱의 합/차 인수분해

📖 핵심 개념

세제곱의 합: a³ + b³ = (a+b)(a²-ab+b²)

세제곱의 차: a³ - b³ = (a-b)(a²+ab+b²)

문제 12의 3차원 버전! 이제 세제곱도 인수분해할 수 있습니다.
8+27 = 2³+3³ = (2+3)(4-6+9) = 5×7 = 35로 확인!
🔍 따라할 예제

문제: x³ + 8 인수분해

1 x³ + 2³ 형태 인식
2 a = x, b = 2
3 (a+b) = (x+2)
4 (a²-ab+b²) = (x²-2x+4)
답:
(x + 2)(x² - 2x + 4)
📝 여러분이 풀 문제

문제: 8x³ - 27 인수분해

1 (___)³ - (___)³
2 a = ___, b = ___
3 (a-b) = _____
4 (a²+ab+b²) = ___________
답:
(_____)(___________)
📚 상세 해설

한글: 8x³=(2x)³, 27=3³. a=2x, b=3. 차 공식: (2x-3)(4x²+6x+9)

English: 8x³=(2x)³, 27=3³. a=2x, b=3. Difference formula: (2x-3)(4x²+6x+9)

복합 인수분해: 치환과 그룹핑

📖 핵심 개념

복잡한 식을 간단한 변수로 치환하여 인수분해합니다.

예: x⁴-5x²+4에서 x²=t로 치환 → t²-5t+4

지금까지 배운 모든 기법을 종합! 치환으로 간단하게 만든 후 인수분해합니다.
x⁴은 (x²)²입니다. x²를 하나의 문자처럼 생각해보세요!
🔍 따라할 예제

문제: x⁴ - 13x² + 36 인수분해

1 t = x²로 치환
2 t² - 13t + 36
3 (t-4)(t-9)
4 (x²-4)(x²-9)
5 (x+2)(x-2)(x+3)(x-3)
📝 여러분이 풀 문제

문제: x⁴ - 10x² + 9 인수분해

1 t = ___로 치환
2 ___________
3 (___)(___)
4 최종: ___________
📚 상세 해설

한글: t=x²로 치환: t²-10t+9=(t-1)(t-9)=(x²-1)(x²-9)=(x+1)(x-1)(x+3)(x-3)

English: Let t=x²: t²-10t+9=(t-1)(t-9)=(x²-1)(x²-9)=(x+1)(x-1)(x+3)(x-3)

비에타의 정리: 근과 계수의 관계

📖 핵심 개념

ax²+bx+c=0의 두 근을 α, β라 하면:

• 근의 합: α+β = -b/a

• 근의 곱: αβ = c/a

이것이 비에타의 정리입니다. 근을 직접 구하지 않고도 합과 곱을 알 수 있습니다!

문제 8의 원리! (x-α)(x-β)=x²-(α+β)x+αβ에서 유도됩니다.
x²-5x+6=0의 근의 합은 5, 곱은 6. 실제로 근 2,3을 구하면 2+3=5, 2×3=6!
🔍 따라할 예제

문제: 2x² - 8x + 6 = 0의 근의 합과 곱

1 a = 2, b = -8, c = 6
2 근의 합 = -b/a = -(-8)/2 = 4
3 근의 곱 = c/a = 6/2 = 3
답:
합: 4, 곱: 3
📝 여러분이 풀 문제

문제: 3x² + 12x - 15 = 0의 근의 합과 곱

1 a = ___, b = ___, c = ___
2 근의 합 = _____
3 근의 곱 = _____
📚 상세 해설

한글: a=3, b=12, c=-15. 근의 합=-12/3=-4, 근의 곱=-15/3=-5

English: a=3, b=12, c=-15. Sum of roots=-12/3=-4, Product of roots=-15/3=-5

심화: 이 방정식을 인수분해하면 3(x²+4x-5)=3(x+5)(x-1). 근은 -5와 1이고, 합=-4, 곱=-5 확인!

대칭식과 인수분해

📖 핵심 개념

대칭식: x와 y를 바꿔도 같은 식 (예: x²+y², xy, x+y)

기본 대칭식 s=x+y, p=xy로 모든 대칭식을 표현할 수 있습니다.

예: x²+y² = (x+y)² - 2xy = s² - 2p

문제 16의 응용! 근의 합과 곱으로 다양한 식의 값을 구할 수 있습니다.
x+y=5, xy=6일 때, x²+y²=(5)²-2(6)=25-12=13
🔍 따라할 예제

문제: x+y=7, xy=10일 때 x³+y³의 값

1 x³+y³ = (x+y)³ - 3xy(x+y)
2 = 7³ - 3(10)(7)
3 = 343 - 210
답:
133
📝 여러분이 풀 문제

문제: x+y=6, xy=8일 때 x⁴+y⁴의 값

1 먼저 x²+y² = _____
2 x⁴+y⁴ = (x²+y²)² - 2(xy)²
3 = ___² - 2(___)²
답:
_____
📚 상세 해설

한글: x²+y²=(x+y)²-2xy=36-16=20. x⁴+y⁴=(x²+y²)²-2(xy)²=400-128=272

English: x²+y²=(x+y)²-2xy=36-16=20. x⁴+y⁴=(x²+y²)²-2(xy)²=400-128=272

분수식의 인수분해와 약분

📖 핵심 개념

분자와 분모를 각각 인수분해한 후 공통인수를 약분합니다.

주의: 약분 전에 정의역을 확인해야 합니다!

모든 인수분해 기법의 종합 응용! 복잡한 분수식도 단순화할 수 있습니다.
(x²-4)/(x-2) = (x+2)(x-2)/(x-2) = x+2 (단, x≠2)
🔍 따라할 예제

문제: (x³-8)/(x²-4) 간단히

1 분자: x³-8 = (x-2)(x²+2x+4)
2 분모: x²-4 = (x+2)(x-2)
3 (x-2) 약분
답:
(x²+2x+4)/(x+2), x≠±2
📝 여러분이 풀 문제

문제: (x⁴-1)/(x³-x) 간단히

1 분자: x⁴-1 = ___________
2 분모: x³-x = ___________
3 약분 후: ___________
📚 상세 해설

한글: 분자: (x²+1)(x+1)(x-1), 분모: x(x+1)(x-1). 약분: (x²+1)/x, x≠0,±1

English: Numerator: (x²+1)(x+1)(x-1), Denominator: x(x+1)(x-1). After canceling: (x²+1)/x, x≠0,±1

복소수와 인수분해 (i = √-1)

📖 핵심 개념

복소수 i = √-1을 사용하면 x²+1도 인수분해 가능!

x²+1 = x²-(-1) = x²-i² = (x+i)(x-i)

복소수 범위에서는 모든 다항식이 1차식의 곱으로 분해됩니다.

문제 12의 확장! 실수에서 불가능했던 인수분해가 복소수에서 가능해집니다.
i²=-1이므로, x²+4 = x²-(-4) = x²-(2i)² = (x+2i)(x-2i)
🔍 따라할 예제

문제: x² + 9 복소수 인수분해

1 x² + 9 = x² - (-9)
2 = x² - 9i²
3 = x² - (3i)²
답:
(x + 3i)(x - 3i)
📝 여러분이 풀 문제

문제: x² + 16 복소수 인수분해

1 x² + 16 = x² - ___
2 = x² - (___)²
답:
(_____)(_____)
📚 상세 해설

한글: x²+16 = x²-(-16) = x²-16i² = x²-(4i)² = (x+4i)(x-4i)

English: x²+16 = x²-(-16) = x²-16i² = x²-(4i)² = (x+4i)(x-4i)

복소수 세제곱근과 고차 인수분해

📖 핵심 개념

1의 세제곱근: 1, ω, ω² (여기서 ω = (-1+i√3)/2)

x³-1 = (x-1)(x-ω)(x-ω²)

오일러 공식: e^(iθ) = cos(θ) + i·sin(θ)

모든 개념의 정점! 복소수를 이용해 고차 다항식을 완전히 인수분해합니다.
x³+1 = x³-(-1) = x³-(e^(iπ))³에서 세 개의 복소수 근이 나옵니다.
🔍 따라할 예제

문제: x³ + 1 완전 인수분해

1 실수 인수: x³+1 = (x+1)(x²-x+1)
2 x²-x+1의 복소수 근:
3 x = (1±i√3)/2
답:
(x+1)(x-(1+i√3)/2)(x-(1-i√3)/2)
📝 여러분이 풀 문제

문제: x⁴ + 4 복소수 인수분해

힌트 Sophie Germain: x⁴+4 = x⁴+4x²+4-4x²
1 = (x²+2)² - _____
2 실수: (_____)(_____)
3 복소수까지: ___________
📚 상세 해설

한글: x⁴+4 = (x²+2)²-(2x)² = (x²+2x+2)(x²-2x+2). 복소수: (x+1±i)의 곱

English: x⁴+4 = (x²+2)²-(2x)² = (x²+2x+2)(x²-2x+2). Complex: product of (x+1±i)

최종: x⁴+4 = (x-1-i)(x-1+i)(x+1-i)(x+1+i)
