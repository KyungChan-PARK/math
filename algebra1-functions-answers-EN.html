Problem 1: Student ID System

Solution:

a) Each student (input) has exactly one ID number (output), and the same student always has the same ID. This one-to-one correspondence defines a function.
b) No. If one student could have multiple IDs, one input would map to multiple outputs, violating the definition of a function.
c) Yes. Since each ID is unique and belongs to exactly one student, the reverse mapping (ID → student) is also a function (actually, it's the inverse function).

Key Concept:

A function requires each input to have exactly one output. Consistency and uniqueness are essential.

Problem 18: Solar Panel Efficiency

Solution:

a) Efficiency function:
E(T) = { 100 if T ≤ 77°F 100 - 0.4(T-77) if T > 77°F }
b) Power at 95°F:
E(95) = 100 - 0.4(95-77) = 100 - 0.4(18) = 92.8% Power = 400W × 0.928 = 371.2W
c) Temperature for 90% efficiency:
90 = 100 - 0.4(T-77) 10 = 0.4(T-77) T-77 = 25 T = 102°F
d) Daily energy with T(h) = 77 + 20sin((h-6)π/12):
This requires integration of power function over 24 hours. Peak temperature occurs at h=12 (noon) with T=97°F, minimum at h=0 or h=24 with T=57°F. Average daily energy ≈ 8.5 kWh
Problem 30: FIRE Formula

Solution:

a) FIRE target:
Target = $72,000 × 25 = $1,800,000

P(t) = 150,000(1.08)^t + 36,000((1.08)^t - 1)/0.08 where t is in years

New annual savings = $42,000 New timeline: t ≈ 15.2 years Time saved: 1.5 years
