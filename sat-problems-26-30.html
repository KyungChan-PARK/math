바이러스 확산 모델링

🦠 상황 설명

감염병 연구소가 새로운 바이러스의 확산을 모델링하고 있습니다:

확산 모델:
• 기본 모델: N(t) = N₀ × 2^(t/3) (t: 일, N: 감염자 수)
• 방역 효과: R(t) = N₀ × 2^(t/3) × 0.8^(t/7)
• 치료제 투입: T(t) = N₀ × 2^(t/3) × (1 - 0.1t)

초기 감염자 N₀ = 100명이고, 정부는 "7일 내 감염자 1000명 미만 유지"를 목표로 합니다.
각 시나리오별 7일째 감염자 수를 계산하고, 어떤 대응이 가장 효과적인지 분석하시오. 또한 지수함수가 전염병 확산을 모델링하는 이유를 설명하시오.

📖 상세 설명과 사고 과정

1단계: 각 모델별 7일 후 계산

기본 모델 (아무 조치 없음):
N(7) = 100 × 2^(7/3) = 100 × 2^2.33 ≈ 100 × 5.04 = 504명

방역 모델 (사회적 거리두기):
R(7) = 100 × 2^(7/3) × 0.8^(7/7)
= 100 × 5.04 × 0.8
= 403명

치료제 모델:
T(7) = 100 × 2^(7/3) × (1 - 0.7)
= 100 × 5.04 × 0.3
= 151명

결과: 모든 방법이 목표 달성! 치료제가 가장 효과적

소리의 데시벨과 로그 스케일

🔊 상황 설명

음향 엔지니어가 콘서트홀을 설계하고 있습니다:

소리 강도 공식:
• 데시벨: dB = 10 × log₁₀(I/I₀)
• I₀ = 10^(-12) W/m² (최소 가청 강도)

주요 소음원:
• 일반 대화: 60 dB
• 도로 소음: 80 dB
• 록 콘서트: 110 dB
• 제트기: 130 dB

청력 보호 기준은 "8시간 노출 시 85 dB 이하"입니다.
각 소음원의 실제 강도(W/m²)를 계산하고, 두 소리가 합쳐질 때의 데시벨을 구하시오. 또한 로그 스케일이 청각 인지에 적합한 이유를 설명하시오.

📖 상세 설명과 사고 과정

1단계: dB를 실제 강도로 변환

dB = 10 × log₁₀(I/I₀)
I = I₀ × 10^(dB/10)

일반 대화 (60 dB):
I = 10^(-12) × 10^6 = 10^(-6) W/m²

도로 소음 (80 dB):
I = 10^(-12) × 10^8 = 10^(-4) W/m²

록 콘서트 (110 dB):
I = 10^(-12) × 10^11 = 10^(-1) W/m²

20 dB 차이 = 100배 강도 차이!
2단계: 소리의 합성

두 개의 60 dB 소리가 합쳐지면?
I_total = 2 × 10^(-6) W/m²
dB_total = 10 × log₁₀(2 × 10^6)
= 10 × (log₁₀2 + 6)
= 10 × (0.301 + 6)
= 63 dB

놀라운 사실: 60 dB + 60 dB = 63 dB (120 dB가 아님!)
선형 vs 로그 스케일
청각 손상 영역
노출 시간 한계
Weber-Fechner 법칙
감각 = k × log(자극)
인간의 모든 감각이 로그 스케일!

• 청각: 데시벨 (dB)
• 시각: 등급 (magnitude)
• 지진: 리히터 규모
• pH: 수소이온농도

복리 투자와 금융 함수

💰 상황 설명

금융 교육 앱이 다양한 투자 상품을 비교합니다:

투자 옵션:
• 예금: A = P(1 + 0.03)^t
• 주식: A = P × e^(0.08t) × (1 + 0.3sin(2πt))
• 암호화폐: A = P × 2^(t/2) × 0.5^(floor(t/3))
• 연금: A = P × t × 1.05^t

초기 투자금 P = 1000만원, 투자 기간 10년을 고려합니다.
각 투자 옵션의 10년 후 가치를 계산하고, 위험-수익 프로파일을 분석하시오. 또한 복리의 "72의 법칙"을 설명하고 검증하시오.

📖 상세 설명과 사고 과정

1단계: 10년 후 가치 계산

예금 (안정형):
A = 1000 × (1.03)^10 = 1000 × 1.344 = 1344만원
수익률: 34.4%

주식 (변동형):
A = 1000 × e^0.8 × (1 + 0) = 1000 × 2.226 = 2226만원
수익률: 122.6% (변동성 무시 시)

암호화폐 (고위험):
A = 1000 × 2^5 × 0.5^3 = 1000 × 32 × 0.125 = 400만원
수익률: -60% (폭락!)

연금 (혼합형):
A = 1000 × 10 × 1.05^10 = 10000 × 1.629 = 16,290만원
수익률: 1529% (매년 추가 납입 가정)
2단계: 72의 법칙 검증

72의 법칙: 원금 2배까지 걸리는 시간 ≈ 72/이자율(%)

예금 3%: 72/3 = 24년
실제: (1.03)^t = 2 → t = log2/log1.03 = 23.4년 ✓

주식 8%: 72/8 = 9년
실제: e^(0.08t) = 2 → t = ln2/0.08 = 8.66년 ✓

오차 < 5% → 실용적으로 정확!

머신러닝 손실 함수 최적화

🤖 상황 설명

AI 연구팀이 이미지 분류 모델을 훈련시키고 있습니다:

손실 함수들:
• MSE: L₁(x) = (y - f(x))²
• Cross-Entropy: L₂(x) = -y×log(f(x))
• Huber: L₃(x) = {0.5x² (|x|≤1), |x|-0.5 (|x|>1)}

학습률 스케줄:
• 고정: η(t) = 0.01
• 지수 감소: η(t) = 0.1 × 0.95^t
• 코사인 어닐링: η(t) = 0.05(1 + cos(πt/100))

모델은 100 epoch 동안 훈련됩니다.
각 손실 함수의 특성을 분석하고, 최적 학습률 스케줄을 선택하시오. 또한 이러한 함수들이 AI 학습에서 갖는 의미를 설명하시오.

📖 상세 설명과 사고 과정

1단계: 손실 함수 특성

MSE (Mean Squared Error):
• 이차 함수: 큰 오류에 민감
• 미분: 2(y - f(x)) → 선형 기울기
• 용도: 회귀 문제

Cross-Entropy:
• 로그 함수: 확률적 해석
• 미분: -y/f(x) → 급격한 기울기
• 용도: 분류 문제

Huber Loss:
• 조각 함수: 이상치에 강건
• MSE + MAE 혼합
• 용도: 노이즈 많은 데이터
손실 함수 비교
학습률 스케줄
수렴 과정
경사하강법의 수학
θ_new = θ_old - η × ∇L(θ)
• θ: 모델 파라미터
• η: 학습률 (step size)
• ∇L: 손실 함수의 기울기

너무 큰 η → 발산
너무 작은 η → 느린 수렴
적응적 η → 최적 경로!
2단계: 최적 조합 선택

이미지 분류 태스크:
• 손실 함수: Cross-Entropy (확률적 출력)
• 학습률: 코사인 어닐링
• 이유: 주기적 탈출로 더 나은 최솟값 탐색

실험 결과:
• 고정 η: 92% 정확도, 평탄 구간 정체
• 지수 감소: 94% 정확도, 후반 미세조정
• 코사인: 96% 정확도, 지역 최솟값 탈출

양자 컴퓨팅과 확률 진폭 함수

⚛️ 상황 설명

양자 컴퓨터 연구소에서 큐비트 상태를 분석합니다:

큐비트 상태:
|ψ⟩ = α|0⟩ + β|1⟩ (α² + β² = 1)

측정 확률:
P(0) = |α|² = cos²(θ/2)
P(1) = |β|² = sin²(θ/2)

양자 게이트:
• Hadamard: H = (1/√2)[1, 1; 1, -1]
• Pauli-X: X = [0, 1; 1, 0]
• Phase: S = [1, 0; 0, i]

5 큐비트 시스템(2^5 = 32 상태)을 제어합니다.
양자 중첩 상태의 확률 분포를 계산하고, 고전 컴퓨터와의 계산 능력 차이를 분석하시오. 또한 지수 함수가 양자 우월성을 설명하는 이유를 논하시오.

📖 상세 설명과 사고 과정

1단계: 양자 중첩의 수학

1 큐비트:
|ψ⟩ = (1/√2)|0⟩ + (1/√2)|1⟩
P(0) = P(1) = 1/2 (균등 중첩)

2 큐비트:
|ψ⟩ = (1/2)(|00⟩ + |01⟩ + |10⟩ + |11⟩)
4개 상태 동시 존재

5 큐비트:
2^5 = 32개 상태 동시 처리
고전: 32번 계산
양자: 1번 계산!
상태 공간 증가
확률 진폭

2단계: 실용적 응용
