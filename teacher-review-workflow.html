문제 #1: 함수 변환

문제: 함수 \(f(x) = x^2\)가 주어졌을 때, \(g(x) = 2f(x-3) + 1\)의 꼭짓점 좌표는?

선택지:

  1. (3, 1) ✓
  2. (-3, 1)
  3. (3, -1)
  4. (-3, -1)

스캐폴딩:

  • Level 1: 변환 순서를 생각해보세요
  • Level 2: x-3은 오른쪽 이동을 의미합니다
  • Level 3: 단계별로 변환을 적용하세요
  • Level 4: 꼭짓점 (0,0) → (3,0) → (3,0) → (3,1)
  • Level 5: 완전한 해설과 그래프 제공
