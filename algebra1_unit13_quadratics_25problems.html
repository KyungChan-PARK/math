근의 공식 도입

SAT 필수
📖 이차방정식의 근의 공식

$ax^2 + bx + c = 0$의 해:

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

🔍 예제

문제: $2x^2 + 5x - 3 = 0$

1 계수 확인: $a = 2$, $b = 5$, $c = -3$
2 판별식: $b^2 - 4ac = 25 - 4(2)(-3) = 49$
3 $x = \frac{-5 \pm \sqrt{49}}{4} = \frac{-5 \pm 7}{4}$
4 $x = \frac{1}{2}$ 또는 $x = -3$

판별식과 해의 개수

📖 판별식 $D = b^2 - 4ac$

• $D > 0$: 서로 다른 두 실근

• $D = 0$: 중근 (하나의 실근)

• $D < 0$: 실근 없음 (복소수 해)

판별식은 포물선이 x축과 만나는 점의 개수를 알려줍니다!
🔍 예제

문제: $x^2 - 4x + 4 = 0$의 해의 개수는?

1 $a = 1$, $b = -4$, $c = 4$
2 $D = (-4)^2 - 4(1)(4) = 16 - 16 = 0$
3 $D = 0$이므로 중근
4 해: $x = \frac{4}{2} = 2$ (중근)
💡 $(x - 2)^2 = 0$으로도 확인 가능!
📝 연습

비에타 공식과 근과 계수의 관계

📖 비에타 공식

$ax^2 + bx + c = 0$의 두 근을 $\alpha$, $\beta$라 하면:

• 합: $\alpha + \beta = -\frac{b}{a}$

• 곱: $\alpha \cdot \beta = \frac{c}{a}$

🔍 예제

문제: $x^2 - 5x + 6 = 0$의 두 근의 합과 곱은?

방법 1 비에타 공식
• 합: $-\frac{(-5)}{1} = 5$
• 곱: $\frac{6}{1} = 6$
방법 2 직접 계산
• $(x - 2)(x - 3) = 0$ → 근: 2, 3
• 합: $2 + 3 = 5$ ✓, 곱: $2 \times 3 = 6$ ✓
📝 연습

2 답: $x^2$ ___$x$ ___ $= 0$
