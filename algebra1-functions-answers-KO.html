문제 1: 학생 ID 시스템

해답:

a) 각 학생(입력)은 정확히 하나의 ID 번호(출력)를 가지며, 같은 학생은 항상 같은 ID를 갖습니다. 이러한 일대일 대응이 함수를 정의합니다.
b) 아니요. 한 학생이 여러 ID를 가질 수 있다면, 하나의 입력이 여러 출력으로 매핑되어 함수의 정의를 위반합니다.
c) 예. 각 ID는 고유하고 정확히 한 명의 학생에게 속하므로, 역방향 매핑(ID → 학생)도 함수입니다 (실제로 역함수입니다).

핵심 개념:

함수는 각 입력이 정확히 하나의 출력을 가져야 합니다. 일관성과 유일성이 필수적입니다.

문제 3: 우버 요금 알고리즘

해답:

a) 급증 전:
기본 요금: $2.55 시간 요금: 12 × $0.35 = $4.20 거리 요금: 4.2 × $1.75 = $7.35 소계: $2.55 + $4.20 + $7.35 = $14.10
b) 급증 적용:
최종 요금 = $14.10 × 1.5 = $21.15
c) 정의역: 시간 ≥ 0분 AND 거리 ≥ 0마일
치역: 요금 ≥ $2.55 (최소 기본 요금)

문제 9: 복리 이자

해답:

a) 잔액:
1년: A = 10,000(1.00375)^12 = $10,459.00 2년: A = 10,000(1.00375)^24 = $10,938.07 3년: A = 10,000(1.00375)^36 = $11,438.13

12,000 = 10,000(1.00375)^(12t) 1.2 = (1.00375)^(12t) ln(1.2) = 12t × ln(1.00375) t = 0.1823 ÷ (12 × 0.00374) = 4.06년
c) 3년간 단리:
단리: 10,000 × 0.045 × 3 = $1,350 복리 이익: $1,438.13 차이: 복리가 $88.13 더 많음
문제 18: 태양광 패널 효율

해답:

a) 효율 함수:
E(T) = { 100 (T ≤ 77°F일 때) 100 - 0.4(T-77) (T > 77°F일 때) }
b) 95°F에서 전력:
E(95) = 100 - 0.4(95-77) = 100 - 0.4(18) = 92.8% 전력 = 400W × 0.928 = 371.2W
c) 90% 효율 온도:
90 = 100 - 0.4(T-77) 10 = 0.4(T-77) T-77 = 25 T = 102°F
d) T(h) = 77 + 20sin((h-6)π/12)에서 일일 에너지:
24시간 동안 전력 함수의 적분이 필요합니다. 최고 온도는 h=12(정오)에 T=97°F, 최저는 h=0 또는 h=24에 T=57°F입니다. 평균 일일 에너지 ≈ 8.5 kWh
문제 30: FIRE 공식

해답:

a) FIRE 목표:
목표 = $72,000 × 25 = $1,800,000
b) 포트폴리오 성장 모델:
P(t) = 150,000(1.08)^t + 36,000((1.08)^t - 1)/0.08 (t는 연 단위)
c) FIRE 도달 시간:
P(t) = 1,800,000으로 설정하고 풀면: t ≈ 16.7년
d) 월 $500 추가 시 (총 $3,500):
새 연간 저축 = $42,000 새 시간표: t ≈ 15.2년 절약된 시간: 1.5년

저축률의 작은 증가는 복리 성장으로 인해 재정 독립까지의 시간을 크게 단축시킬 수 있습니다.
