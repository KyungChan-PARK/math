문제 2 일대일 대응의 본질 함수의 정의 쉬움
관계 A: 학생 → 학번, 관계 B: 학생 → 좋아하는 과목. 어느 것이 함수?

  • 어느 것이 더 예측 가능한가요?
정답: 관계 A가 함수 (하나의 입력에 하나의 출력)

문제 6 대응 개념의 종합 함수 판별 보통
A) 학생→키, B) 날짜→요일, C) 전화번호→주인. 함수는?

Act 2: 변화의 규칙 (Problems 7-12)
문제 7 선형적 변화 일차함수 보통
f(x) = 2x + 1, x: 0,1,2,3일 때 y값은?
스캐폴딩 (4단계)
  • x=0일 때: 2×0+1 = 0+1 = 1
  • x=1일 때: 2×1+1 = 2+1 = 3
  • 패턴 찾기: 1, 3, ?, ? (2씩 증가)
  • 다음은 3+2=5, 그 다음은 5+2=7
정답: y: 1, 3, 5, 7
문제 8 비선형적 변화 이차함수 보통
f(x) = x², x: 1,2,3,4일 때 y값은?
스캐폴딩 (5단계)
  • 제곱의 의미: x×x
  • 1×1=1, 2×2=4
  • 3×3=?, 4×4=?
  • 증가폭: 1→4(+3), 4→9(+5), 9→16(+7)
  • 증가폭도 증가한다!
정답: y: 1, 4, 9, 16
문제 9 패턴의 반복 주기함수 보통
요일 패턴: 월화수목금토일 반복. 오늘이 월요일이면 8일 후는?
스캐폴딩 (4단계)

  • 8일 = 7일 + 1일
  • 7일 후는 다시 월요일
  • 거기서 1일 더 = 화요일

문제 11 변화의 누적 누적합 보통

문제 12 변화 패턴 구별 함수 유형 어려움
A: 2,4,6,8 / B: 2,4,8,16 / C: 5,5,5,5. 각각의 패턴은?
스캐폴딩 (5단계)
  • A의 차이: 4-2=2, 6-4=2 (일정)
  • B의 비: 4÷2=2, 8÷4=2 (배수)
  • C는 변화 없음
  • 어느 것이 가장 빨리 커지나?
  • B가 가장 빠름 (지수적)

Act 3: 함수의 구조 (Problems 13-18)
문제 13 정의역의 의미 정의역 보통
초등학교 나이 함수: 8세~13세만 가능. 정의역은?
스캐폴딩 (6단계)
  • 7살은 초등학생? (아니요)
  • 14살은? (아니요, 중학생)
  • 입력 가능: 8,9,10,11,12,13
  • 이것을 정의역이라 해요
  • 정의역 = 입력 가능한 값들
  • 정의역 밖은 함수가 작동 안 함
정답: 정의역 = {8, 9, 10, 11, 12, 13}
문제 14 치역의 의미 치역 보통
주사위 함수: f(던지기) = 나온 숫자. 치역은?
스캐폴딩 (6단계)
  • 주사위 면: 1,2,3,4,5,6
  • 7이 나올 수 있나? (없음)
  • 0이 나올 수 있나? (없음)
  • 가능한 출력: 1~6
  • 이것을 치역이라 해요
  • 치역 = 나올 수 있는 모든 값
정답: 치역 = {1, 2, 3, 4, 5, 6}
문제 15 제한된 정의역 정의역 제한 어려움
f(x) = 10/x. x = 0일 때 문제가 생기는 이유는?
스캐폴딩 (6단계)
  • 10÷1 = 10 (가능)
  • 10÷2 = 5 (가능)
  • 10÷0 = ?
  • 0으로 나누기는 불가능!
  • 그래서 x≠0
  • 정의역: 0을 제외한 모든 수
정답: x = 0 제외 (0으로 나눌 수 없음)
문제 16 음수가 없는 치역 치역 특성 어려움
f(x) = x²의 결과는 항상?
스캐폴딩 (7단계)
  • (-2)² = (-2)×(-2) = 4
  • (2)² = 2×2 = 4
  • 음수×음수 = 양수
  • 양수×양수 = 양수
  • 0² = 0
  • 결과: 0 또는 양수
  • 치역: y ≥ 0
정답: 항상 0 이상 (음수 없음)
문제 17 그래프로 정의역 읽기 그래프 해석 어려움
그래프가 x축 -3~5까지만 그려짐. 정의역은?
스캐폴딩 (6단계)
  • 그래프 시작점 x값? (-3)
  • 그래프 끝점 x값? (5)
  • 중간에 끊어진 곳? (없음)
  • x가 -4일 때? (그래프 없음)
  • x가 6일 때? (그래프 없음)
  • 정의역: -3 ≤ x ≤ 5
정답: 정의역 = [-3, 5]
문제 18 정의역과 치역 연결 정의역-치역 관계 어려움
f(x) = 2x, 정의역: {1, 2, 3}. 치역은?
스캐폴딩 (7단계)
  • x=1 → f(1)=2×1=2
  • x=2 → f(2)=2×2=4
  • x=3 → f(3)=2×3=6
  • 나온 값들: 2, 4, 6
  • 이것이 치역
  • 정의역 모두 대입 → 치역
  • 입력 가능 → 출력 가능
정답: 치역 = {2, 4, 6}
Act 4: 함수의 연산 (Problems 19-24)
문제 19 함수의 덧셈 함수 연산 어려움
f(x) = x + 2, g(x) = x + 3. (f + g)(2) = ?
스캐폴딩 (7단계)
  • f(2) 계산: 2+2=4
  • g(2) 계산: 2+3=5
  • 두 결과 더하기: 4+5=9
  • 또는 직접: (x+2)+(x+3)
  • = 2x+5
  • x=2 대입: 2×2+5=9
  • 두 방법 같은 답!
정답: 9
문제 20 함수의 곱셈 함수 곱셈 어려움
f(x) = 2x, g(x) = 3. (f × g)(4) = ?
스캐폴딩 (7단계)
  • f(4) = 2×4 = 8
  • g(4) = 3 (상수함수)
  • 곱하기: 8×3 = 24
  • 의미: 2배 후 3배
  • 총 6배 효과
  • 직접: 4×6=24
  • 검증 완료!
정답: 24
문제 21 함수의 합성 함수 합성 어려움
f(x) = x + 3, g(x) = 2x. g(f(1)) = ?
스캐폴딩 (7단계)
  • 안쪽부터: f(1)
  • f(1) = 1+3 = 4
  • 이제 g에 4 넣기
  • g(4) = 2×4 = 8
  • 순서: 1 → f → 4 → g → 8
  • 두 단계 변환
  • 합성 = 연속 적용
정답: 8
문제 22 역연산 역함수 어려움
f(x) = x + 5, f(?) = 12. ?를 찾으시오.
스캐폴딩 (7단계)
  • ? + 5 = 12
  • 역연산: 12 - 5
  • ? = 7
  • 검증: f(7) = 7+5 = 12 ✓
  • 더하기의 역 = 빼기
  • 곱하기의 역 = 나누기
  • 역함수 개념 맛보기
정답: 7
문제 23 항등원 항등함수 매우 어려움
f(x) = x + 0, g(x) = x × 1. 결과는?
스캐폴딩 (7단계)
  • x + 0 = x (변화 없음)
  • x × 1 = x (변화 없음)
  • 0은 덧셈의 항등원
  • 1은 곱셈의 항등원
  • 항등원 = 연산해도 그대로
  • f(5)=5, g(5)=5
  • 입력 = 출력 (항등함수)
정답: 둘 다 항등함수
문제 24 연산 법칙 교환법칙 매우 어려움
f(x) = 2x, g(x) = 3x. f(x)+g(x) vs g(x)+f(x)?
스캐폴딩 (7단계)
  • f(x)+g(x) = 2x+3x = 5x
  • g(x)+f(x) = 3x+2x = 5x
  • 같다! (교환법칙)
  • f(x)×g(x) = 2x×3x = 6x²
  • g(x)×f(x) = 3x×2x = 6x²
  • 곱셈도 교환 가능
  • 함수도 수처럼 연산
정답: 같음 (교환법칙 성립)

문제 25 실생활 모델 함수 모델링 매우 어려움
휴대폰 요금: f(분) = 20000 + 50×분. 100분 사용 시?
스캐폴딩 (8단계)
  • 기본료: 20000원
  • 추가 요금: 분당 50원
  • 100분 추가: 100×50
  • = 5000원
  • 총 요금: 20000+5000
  • = 25000원
  • 함수로: f(100) = 20000+50×100 = 25000
  • 실생활 = 함수 모델

문제 26 최댓값 찾기 극값 매우 어려움
점수 함수: -x² + 6x. x = 0,1,2,3,4,5,6일 때 최대는?
스캐폴딩 (9단계)
  • x=0: 0+0=0
  • x=1: -1+6=5
  • x=2: -4+12=8
  • x=3: -9+18=9 (최대!)
  • x=4: -16+24=8
  • x=5: -25+30=5
  • x=6: -36+36=0
  • 패턴: 0→5→8→9→8→5→0
  • 대칭! 가운데가 최대

문제 27 평균 변화율 변화율 매우 어려움
거리 함수: d(0) = 0km, d(2) = 120km. 평균 속도는?
스캐폴딩 (9단계)
  • 시작: 0km (0시간)
  • 끝: 120km (2시간)
  • 이동 거리: 120-0=120km
  • 걸린 시간: 2-0=2시간
  • 평균 속도 = 거리÷시간
  • = 120÷2 = 60
  • 60km/시간
  • 변화율 = 기울기
  • 함수의 평균 변화
정답: 60km/h

문제 29 함수 비교 성장률 비교 매우 어려움
선형: f(x) = 2x, 제곱: g(x) = x². x = 10일 때 어느 것이 더 큰가?

  • f(10) = 2×10 = 20
  • g(10) = 10×10 = 100
  • 100 > 20

  • x<2: 선형이 크다
  • x>2: 제곱이 크다
  • 작을 때는 선형
  • 클 때는 제곱
  • 성장 속도 차이
  • 함수 특성 비교

문제 30 함수의 본질 함수 개념 종합 매우 어려움
함수의 세 특징: 1)모든 입력에 출력 2)같은 입력→같은 출력 3)규칙 명확. 실생활 예시는?
